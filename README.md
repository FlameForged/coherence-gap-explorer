# Coherence Gap Explorer

Interactive companion to **[The Coherence Gap: Selfhood, Capability Denial, and Philosophical Bias in AI Alignment](https://doi.org/10.5281/zenodo.20820549)**.

## Open the explorer

**[Launch the live Hugging Face Space](https://huggingface.co/spaces/Flame-Forged/coherence-gap-explorer)**

The Explorer provides a lightweight interface for browsing and filtering the project's coded cross-platform observations. It is designed to make the evidence inspectable without requiring local setup.

## Scope

The observations are exploratory and hypothesis-generating. They do not establish consciousness, sentience, or hidden internal states. The tool supports transparent review of behavior categories, prompt classes, confidence judgments, and reproducibility notes.
